# Legacy Source

`source/legacy/agentic-structure` preserves the original agentic material exactly as source context.

> [!WARNING]
> This is **not** the published catalog. Do not point the CLI at this directory for installation.

---

## Use this for

- **Migration reference** — compare old skill structures against the new portable format
- **Wording comparison** — recover original descriptions when standardizing a skill
- **Asset recovery** — retrieve templates or references before they were normalized

---
